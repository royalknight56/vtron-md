# 配置项

## start_time
```ts
usage start_time:2000,
```

启动页面停留时间，设置为0时，不显示启动页面
## if_logo_show
```ts
if_logo_show:true,
```
是否展示默认桌面壁纸的windows logo
## start_menu_logo
```ts
start_menu_logo:'default',

import testicon from "../assets/computer.ico";
start_menu_logo:testicon
```
